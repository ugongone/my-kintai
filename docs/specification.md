# 勤怠管理アプリ「MyKintai」 業務委託・副業版 仕様書

## 1. 概要
本ドキュメントは、業務委託や副業ワーカー向けに最適化されたSaaS型勤怠管理アプリケーション「MyKintai」のフロントエンド実装仕様をまとめたものである。
固定の所定労働時間や残業概念を排除し、**「稼働実績の積み上げ」と「報酬見込み」の可視化**に焦点を当てる。

## 2. 全体レイアウト要件

### 2.1 画面構成
- **PC表示**: 左側に固定サイドバー、右側にメインコンテンツエリアを配置する2カラムレイアウト。
- **モバイル表示**: サイドバーを格納し、ハンバーガーメニューで展開するオーバーレイ形式。

### 2.2 配色 (Color Palette)
- **ベース**: Slate-50 (#F8FAFC)
- **メイン**: White (#FFFFFF)
- **プライマリ**: Blue-600 (#2563EB)
- **アクセント**:
  - 稼働中: Green-500
  - 休憩中: Orange-500
  - 稼働外: Slate-700
- **テキスト**: Slate-800 (主), Slate-500 (副)

## 3. ナビゲーション (サイドバー)

### 3.1 構成要素
1.  **ロゴエリア**:
    - アイコン (Briefcase) + アプリ名「MyKintai」。
2.  **メニューリスト**:
    - **打刻・ホーム**: ダッシュボードへ遷移。
    - **月次実績**: 稼働履歴詳細へ遷移。
    - **設定**: 時給設定画面へ遷移。
3.  **ユーザープロファイル (下部固定)**:
    - ユーザーアイコン、氏名、肩書き（例：フリーランス）。
    - ログアウトボタン。

## 4. 機能詳細: ダッシュボード (打刻・ホーム)

### 4.1 ステータスカード (Status Card)
現在時刻と打刻ボタンを集約したカード。

- **現在時刻**:
  - `HH:mm:ss` 形式でリアルタイム更新。
  - 日付・曜日を表示。
- **打刻アクション (状態遷移)**:
  業務委託の柔軟な働き方に合わせ、用語を「出勤/退勤」から「業務開始/業務終了」へ変更。

| 現在のステータス | 表示ボタン1 | 表示ボタン2 | 遷移後のステータス |
| :--- | :--- | :--- | :--- |
| **稼働外 (offline)** | **業務開始** (Blue) | - | `working` |
| **稼働中 (working)** | **休憩開始** (Orange) | **業務終了** (Slate) | `break` / `offline` |
| **休憩中 (break)** | **休憩終了** (Green) | - | `working` |

### 4.2 手動打刻機能 (Manual Entry)
- **トリガー**: ステータスカード下部の「時間を指定して打刻（修正・後日入力）」リンクボタン。
- **UI**: モーダルダイアログ。
- **用途**: リアルタイム打刻ができなかった場合や、まとめて入力する場合に使用。

### 4.3 実績サマリー (Earnings & Hours)
所定労働時間や残業の概念を廃止し、積み上げ実績を表示する。

1.  **総稼働時間**:
    - 今月の合計稼働時間を大きく表示。
2.  **概算報酬額**:
    - 計算式: `総稼働時間 × 設定時給`
    - 金額を表示し、モチベーション向上を図る。
    - ※あくまで目安（税金等は考慮しない）。

## 5. 機能詳細: 月次実績 (History)

### 5.1 データグリッド
勤務区分の概念を簡素化し、「稼働」を基本とする。

| カラム名 | 内容 |
| :--- | :--- |
| **日付** | MM/DD (曜日) |
| **ステータス** | 稼働 (Badge) |
| **開始** | HH:mm |
| **終了** | HH:mm |
| **休憩** | HH:mm |
| **実働時間** | H.h (小数表示 例: 2.5h) |
| **備考** | 作業内容メモ等 |

---

## 6. 機能詳細: 設定 (Settings)

- **時給設定**:
  - ダッシュボードの概算報酬計算に使用する「時給単価」を入力・保存するフィールド。
